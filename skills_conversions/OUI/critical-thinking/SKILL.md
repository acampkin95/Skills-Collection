# Critical Thinking

Systematic evaluation of arguments, claims, and reasoning. Covers logical fallacies, cognitive biases, argument analysis frameworks, Socratic questioning, and evidence assessment.

## Core Principles

1. **Separate observation from inference** — What was observed vs. what was concluded
2. **Question assumptions** — Every argument rests on hidden premises
3. **Evaluate evidence quality** — Not all evidence is equal
4. **Consider alternatives** — The best explanation may not be the first one
5. **Quantify uncertainty** — Express confidence levels explicitly

## Argument Analysis

### Toulmin Model

Every argument can be decomposed into six elements:

```
┌──────────────────────────────────────────────────────┐
│ CLAIM (Conclusion)                                    │
│ "X is true / should happen"                          │
│                                                      │
│ supported by                                         │
│                                                      │
│ GROUNDS (Evidence/Data)                              │
│ "Because [data/facts/observations]"                  │
│                                                      │
│ connected by                                         │
│                                                      │
│ WARRANT (Reasoning)                                  │
│ "Since [general principle that connects grounds      │
│  to claim]"                                          │
│                                                      │
│ strengthened by                                      │
│                                                      │
│ BACKING (Support for warrant)                        │
│ "The warrant is reliable because [authority/data]"   │
│                                                      │
│ qualified by                                         │
│                                                      │
│ QUALIFIER (Degree of certainty)                      │
│ "Probably / likely / in most cases"                  │
│                                                      │
│ limited by                                           │
│                                                      │
│ REBUTTAL (Exceptions/counter-arguments)              │
│ "Unless [conditions that would invalidate]"          │
└──────────────────────────────────────────────────────┘
```

### Argument Evaluation Checklist

- [ ] **Claim identified**: Is the main conclusion clearly stated?
- [ ] **Grounds adequate**: Is there sufficient evidence to support the claim?
- [ ] **Warrant valid**: Does the reasoning logically connect grounds to claim?
- [ ] **Backing provided**: Is the warrant itself supported?
- [ ] **Qualifier appropriate**: Is the degree of certainty justified by evidence?
- [ ] **Rebuttals addressed**: Are counter-arguments acknowledged and refuted?

## Logical Fallacies Catalog

### Fallacies of Relevance (Red Herring Family)

| Fallacy | Pattern | Example |
|---------|---------|---------|
| **Ad Hominem** | Attack the person, not the argument | "You can't trust their climate research — they fly private jets" |
| **Ad Hominem (Circumstantial)** | Attack motives/interests | "Of course they support the policy — they'll profit from it" |
| **Tu Quoque** | "You too" — deflect with hypocrisy | "You criticize my driving but you got a ticket last month" |
| **Poisoning the Well** | Pre-emptively discredit source | "Before they speak, remember they were wrong before" |
| **Genetic Fallacy** | Judge by origin, not merit | "That idea came from a blog, so it can't be right" |
| **Guilt by Association** | Link to disliked group/idea | "That policy was also supported by [bad actor]" |
| **Appeal to Authority** | Unqualified authority as evidence | "A celebrity said this diet works" |
| **Appeal to Popularity** | Many believe it, so it's true | "Millions of people use this, so it must be effective" |
| **Appeal to Tradition** | It's always been done this way | "This is how we've done it for 20 years" |
| **Appeal to Novelty** | New means better | "Our new approach is revolutionary — old methods are obsolete" |
| **Appeal to Emotion** | Feelings replace evidence | "Think of the children! We must act now!" |
| **Appeal to Fear** | Scare tactics as persuasion | "If we don't act, catastrophe is inevitable" |

### Fallacies of Causation

| Fallacy | Pattern | Example |
|---------|---------|---------|
| **Post Hoc** | A before B, therefore A caused B | "Stocks rose after the speech, so the speech caused it" |
| **Cum Hoc** | A correlates with B, therefore A causes B | "Ice cream sales and drowning both increase in summer" |
| **Slippery Slope** | A leads to Z without showing intermediate steps | "If we allow X, next we'll have Y, and eventually Z!" |
| **False Cause** | Coincidence presented as causation | "Every time I wear this shirt, my team wins" |
| **Single Cause** | Complex outcome attributed to one cause | "The project failed because of the deadline" |
| **Regression Fallacy** | Ignore natural fluctuation | "After the training, scores improved" (they regressed to mean) |

### Fallacies of Structure

| Fallacy | Pattern | Example |
|---------|---------|---------|
| **Straw Man** | Misrepresent opponent's position | "So you're saying we should just let criminals go free?" |
| **False Dichotomy** | Only two options presented | "Either we cut costs or we go bankrupt" |
| **False Equivalence** | Treat unequal things as equal | "Both sides have valid points" (when evidence strongly favors one) |
| **Begging the Question** | Conclusion assumed in premise | "The Bible is true because it says so in the Bible" |
| **Circular Reasoning** | A proves B, B proves A | "They're trustworthy because they said so, and they're honest" |
| **No True Scotsman** | Redefine terms to exclude counter-examples | "No real programmer would use that tool" |
| **Moving the Goalposts** | Change criteria after evidence presented | "That doesn't count, I meant a different kind of evidence" |
| **Loaded Question** | Presupposition embedded in question | "Have you stopped making that mistake?" |

### Fallacies of Evidence

| Fallacy | Pattern | Example |
|---------|---------|---------|
| **Cherry Picking** | Select favorable data only | "These three studies show it works" (ignoring 20 that don't) |
| **Anecdotal Evidence** | Personal story as proof | "My uncle smoked and lived to 90, so smoking isn't harmful" |
| **Hasty Generalization** | Small sample, broad conclusion | "I met two rude people from there — everyone there is rude" |
| **Texas Sharpshooter** | Draw target around hits | "Look at all these coincidences!" (ignoring the non-coincidences) |
| **Survivorship Bias** | Only count successes | "These successful entrepreneurs dropped out — so should you" |
| **Base Rate Neglect** | Ignore prior probabilities | "The test is 99% accurate and I tested positive!" (ignoring low prevalence) |

## Cognitive Biases

### Decision Biases

| Bias | Description | Mitigation |
|------|-------------|------------|
| **Anchoring** | Over-rely on first piece of information | Seek multiple independent estimates |
| **Confirmation Bias** | Seek/notice confirming evidence, ignore disconfirming | Actively search for counter-evidence |
| **Framing Effect** | Same information, different conclusions based on presentation | Reframe in both positive and negative terms |
| **Sunk Cost** | Continue investing because of past investment | Evaluate only future costs and benefits |
| **Status Quo Bias** | Prefer current state over change | Evaluate options from a blank slate |
| **Loss Aversion** | Losses feel 2x worse than equivalent gains | Evaluate in absolute terms, not gains/losses |
| **Endowment Effect** | Overvalue things we own | Ask "what would I pay to acquire this?" |
| **Decoy Effect** | Irrelevant option changes preference between two others | Remove decoy, re-evaluate the original comparison |

### Information Biases

| Bias | Description | Mitigation |
|------|-------------|------------|
| **Availability Heuristic** | Judge probability by ease of recall | Use base rates and statistics, not memory |
| **Representativeness** | Judge by similarity to prototype | Consider actual probability, not resemblance |
| **Dunning-Kruger** | Low expertise → overconfidence | Seek expert review, acknowledge uncertainty |
| **Illusion of Validity** | Confidence in wrong predictions | Track prediction accuracy over time |
| **Hindsight Bias** | "I knew it all along" after the fact | Document predictions before outcomes |
| **Bandwagon Effect** | Adopt beliefs because others hold them | Evaluate evidence independently |
| **Narrative Fallacy** | Force causal stories onto random events | Test if the story predicts future outcomes |
| **Recency Bias** | Overweight recent events | Use historical data and long-term trends |

### Social Biases

| Bias | Description | Mitigation |
|------|-------------|------------|
| **In-group Bias** | Favor group members' opinions | Evaluate arguments on merit, not source |
| **Authority Bias** | Over-value opinions of authority figures | Check if authority is in the relevant domain |
| **Halo Effect** | One positive trait → assume other positives | Evaluate each trait independently |
| **Fundamental Attribution Error** | Others' actions = character; our actions = situation | Consider situational factors for everyone |
| **Actor-Observer Bias** | Attribute own success to skill, failure to circumstances | Apply same lens to self and others |

## Socratic Questioning

### Six Types of Socratic Questions

| Type | Purpose | Example Questions |
|------|---------|-------------------|
| **Clarification** | Understand the claim | What exactly do you mean by X? Can you give an example? |
| **Probing Assumptions** | Examine hidden premises | What are you assuming here? Is that assumption justified? |
| **Probing Evidence** | Test the grounds | What evidence supports this? How reliable is that source? |
| **Viewpoint & Perspectives** | Consider alternatives | What would someone who disagrees say? Are there other ways to interpret this? |
| **Implications & Consequences** | Follow the logic through | If this is true, what else follows? What are the practical consequences? |
| **Questions about the Question** | Meta-level examination | Why is this question important? What assumptions does the question itself contain? |

### Socratic Questioning Template

```
FOR ANY CLAIM, ASK:

1. CLARITY: What precisely is being claimed?
2. SOURCE: Where does this claim come from?
3. EVIDENCE: What evidence supports it?
4. ASSUMPTIONS: What must be true for this claim to hold?
5. ALTERNATIVES: What other explanations exist?
6. IMPLICATIONS: If true, what follows?
7. COUNTER-EVIDENCE: What evidence contradicts it?
8. CONFIDENCE: How certain can we be, and why?
```

## Bayesian Reasoning Basics

### Core Concept

Update beliefs based on evidence. Prior probability adjusts with new evidence:

```
PRIOR (what you believed before)
    + EVIDENCE (new information)
    = POSTERIOR (updated belief)

Strength of update depends on:
  - How strong the prior was (strong priors need strong evidence to shift)
  - How diagnostic the evidence is (does it distinguish between hypotheses?)
```

### Base Rate Neglect (Most Common Error)

```
SCENARIO: A test for a rare disease is 99% accurate.
          Disease prevalence is 1 in 10,000.
          You test positive. What's the probability you have the disease?

INTUITIVE ANSWER: ~99%
CORRECT ANSWER: ~1%

WHY: Out of 10,000 people:
  - 1 actually has the disease → tests positive (true positive)
  - 9,999 don't have it → ~100 test positive (false positives)
  - So 101 test positive, only 1 actually has it → ~1% chance

LESSON: Always consider base rates before updating on evidence.
```

### Practical Bayesian Questions

| Question | Purpose |
|----------|---------|
| What's the base rate? | How common is this event/condition generally? |
| How diagnostic is this evidence? | Would I see this evidence regardless, or only if the hypothesis is true? |
| What would I expect to see if wrong? | What evidence would disconfirm my hypothesis? |
| Am I updating proportionally? | Is my confidence shift proportional to evidence strength? |

## Evidence Hierarchy

### Strength Levels

```
TIER 1 — Strongest Evidence
├── Systematic reviews / meta-analyses
├── Large randomized controlled trials (RCTs)
└── Consensus statements from multiple independent sources

TIER 2 — Strong Evidence
├── Individual RCTs with adequate sample size
├── Large-scale observational studies with controls
└── Replicated experimental findings

TIER 3 — Moderate Evidence
├── Well-designed observational studies
├── Expert consensus (when based on systematic review)
└── Single well-controlled experiment

TIER 4 — Weak Evidence
├── Case studies and case reports
├── Expert opinion (without systematic review)
├── Small uncontrolled studies
└── Personal experience / anecdotes

TIER 5 — Not Evidence
├── Unsupported assertions
├── Traditional practice ("we've always done it this way")
└── Logical arguments without empirical testing
```

### Evaluating Source Quality

| Dimension | High Quality | Low Quality |
|-----------|-------------|-------------|
| **Methodology** | Controlled, reproducible | Uncontrolled, one-off |
| **Sample Size** | Adequate for statistical power | Too small to generalize |
| **Conflict of Interest** | Declared and managed | Undeclared or pervasive |
| **Peer Review** | Independent review | Self-published, no review |
| **Reproducibility** | Replicated by others | Single study, not replicated |

## Argument Mapping

### Structure

```
CENTRAL CLAIM: [Main conclusion]
    │
    ├── REASON 1: [Supporting argument]
    │   ├── Evidence: [data, facts]
    │   ├── Warrant: [why evidence supports reason]
    │   └── Confidence: [high/medium/low]
    │
    ├── REASON 2: [Supporting argument]
    │   ├── Evidence: [data, facts]
    │   ├── Warrant: [why evidence supports reason]
    │   └── Confidence: [high/medium/low]
    │
    ├── OBJECTION 1: [Counter-argument]
    │   └── Response: [rebuttal or concession]
    │
    └── OBJECTION 2: [Counter-argument]
        └── Response: [rebuttal or concession]

OVERALL ASSESSMENT:
  - Net confidence: [high/medium/low]
  - Key uncertainty: [what could change this assessment]
  - Would change mind if: [what evidence would reverse conclusion]
```

## Dialectical Methods

### Steel Man vs. Straw Man

| Approach | Description | When to Use |
|----------|-------------|-------------|
| **Steel Man** | Present the strongest version of the opposing argument | Before critiquing — ensures fair engagement |
| **Iron Man** | Present an even stronger version than the opponent stated | Stress-testing your own position |
| **Straw Man** | Misrepresent and attack a weak version | Avoid this — it's a fallacy |

### Devil's Advocate Protocol

```
1. State your position clearly
2. Generate 3 strongest counter-arguments
3. For each counter-argument:
   a. State it in its strongest form (steel man)
   b. Evaluate the evidence for and against
   c. Rate its strength: strong/moderate/weak
4. Reassess your position considering all counter-arguments
5. State your revised position with confidence level
```

## Claim Strength Assessment

### Rating Scale

| Rating | Description | Standard |
|--------|-------------|----------|
| **Confirmed** | Multiple independent confirmations, no credible contradiction | 3+ Tier 1-2 sources agree |
| **Probable** | Strong evidence supports, minor uncertainty | 2+ Tier 1-2 sources, or strong Tier 2-3 consensus |
| **Plausible** | Consistent with evidence but not yet well-tested | Tier 3 evidence, logical consistency |
| **Undetermined** | Insufficient evidence either way | Limited or contradictory evidence |
| **Doubtful** | Evidence against outweighs evidence for | Significant Tier 1-2 evidence contradicts |
| **False** | Strong evidence contradicts | Multiple Tier 1 sources contradict |

### Quick Assessment Template

```
CLAIM: [statement being evaluated]
EVIDENCE FOR: [list with quality ratings]
EVIDENCE AGAINST: [list with quality ratings]
FALLACIES PRESENT: [any identified]
BIASES TO WATCH: [relevant cognitive biases]
OVERALL RATING: [Confirmed → False scale]
CONFIDENCE: [high/medium/low]
KEY UNCERTAINTY: [what would change this rating]
```

## When to Use

- Evaluating claims and arguments from any source
- Detecting logical fallacies in reasoning or persuasion attempts
- Analyzing research findings or media reports for quality
- Making decisions under uncertainty using Bayesian reasoning
- Conducting structured debates or steel-manning opposing views
- Assessing evidence quality in research or investigative work
- Questioning assumptions in complex problem-solving

## Limitations

- Critical thinking frameworks improve evaluation but don't guarantee correct conclusions
- Fallacy identification requires contextual judgment — not every weak argument is fallacious
- Bayesian reasoning requires base rate data that may not be available
- Cognitive biases affect everyone, including those aware of them — awareness reduces but doesn't eliminate bias
- Some arguments involve values and ethics where logical analysis alone is insufficient
