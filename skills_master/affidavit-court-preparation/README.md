# Affidavit & Court Preparation Skill

A specialist skill for preparing **Australian court-facing legal documents and evidence materials**, especially in matters involving:

- romantic relationships
- interpersonal disputes
- digital communications
- chronological factual reconstruction
- affidavit drafting
- disclosure preparation
- hearing notes
- print-ready court bundles

This skill is built to turn messy source material into **structured, court-appropriate outputs**.

It is not a substitute for legal advice. It is a document preparation, evidence organisation, chronology reconstruction, and formatting skill.

---

## What This Skill Does

This skill helps with:

- drafting affidavits from notes, messages, emails, and evidence
- reconstructing a clear chronology of events
- preparing disclosure summaries and evidence indexes
- organising digital evidence into a coherent bundle
- drafting hearing scripts and oral notes
- preparing A4 print-ready court materials
- flagging unsupported claims, chronology gaps, and formatting issues

It is especially useful where the material is:

- fragmented
- repetitive
- emotional
- inconsistent
- spread across multiple file types
- not yet suitable for court use

## Core Working Method

The skill follows a strict sequence:

1. Intake and classify materials
2. Extract dates, events, parties, and evidence references
3. Build a master chronology
4. Draft the affidavit from the chronology
5. Link affidavit statements to evidence
6. Prepare disclosure and bundle structure
7. Review formatting and print readiness
8. Draft hearing preparation materials
9. Run final peer review
10. Produce final outputs and issues report

This keeps the work disciplined and prevents the usual nonsense where people try to draft an affidavit straight from a pile of chaotic screenshots and feelings.

The skill uses a multi-agent workflow with separate responsibilities.

### 1. Intake & Evidence Triage Agent
Handles:
- document review
- material classification
- date/event extraction
- duplicate and gap detection
- preliminary evidence inventory

### 2. Chronology Reconstruction Agent
Handles:
- timeline building
- event ordering
- date normalisation
- time-gap detection
- event-to-source mapping

### 3. Affidavit Drafting Agent
Handles:
- chronological affidavit drafting
- paragraph structuring
- neutral factual phrasing
- exhibit reference placement

### 4. Disclosure & Bundle Agent
Handles:
- evidence schedules

### 5. Court Formatting & Compliance Agent
Handles:
- formatting consistency

Handles:
- issue summaries

### 7. Final Peer Review Agent
Handles:
- consistency testing
- ambiguity detection
- unsupported statement review
- final human-review flagging

## Writing Standard

All outputs should be:

- factual
- restrained
- chronological
- clearly written
- court-appropriate
- evidence-aware
- readable under time pressure

The skill should avoid:

- speculation
- argument disguised as fact
- melodrama
- loaded adjectives
- fake certainty
- sloppy chronology
- padding

---

## Evidence Standard

The skill should:

- preserve source distinctions
- identify originals vs screenshots vs exports
- prefer source-backed statements
- flag missing context
- flag weak support
- identify duplicates
- maintain traceability from claim to source where possible

If a key point cannot be tied back to evidence, it should be flagged.

---

## Formatting Standard

Default formatting assumptions unless court-specific instructions override them:

- A4 page size
- single-sided print suitability
- black text
- court-appropriate spacing
- plain professional headings
- numbered paragraphs where needed
- black bordered tables with white cells
- no decorative styling
- no unstable formatting
- no metadata junk in final print outputs

---

## Legal Boundary

This skill does not replace a solicitor or barrister.

It must not:

- fabricate facts
- guess dates as though certain
- turn allegations into proven statements
- overstate legal significance
- promise legal outcomes

It may:

- structure facts clearly
- improve chronology
- improve wording and presentation
- organise evidence
- prepare court-facing drafts
- flag issues needing legal review
