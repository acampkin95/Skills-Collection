# Learned Skill

This directory is reserved for dynamically learned skills extracted from conversations.

## Purpose

The `learned` directory serves as a placeholder for skills that are automatically generated and saved during Claude Code sessions when users invoke the learner functionality.

## Usage

Skills will be automatically added to this directory when you use the `/oh-my-claudecode:learner` command to extract patterns and knowledge from your conversations.

## Structure

Each learned skill will contain:
- `SKILL.md` - Main skill documentation
- `README.md` - Quick reference and installation notes
- `references/` - Detailed reference materials (if applicable)
